# 条件筛选统计分析功能说明

## 功能概述

在压力测试执行时，系统会自动根据设置的X变量压力范围，从蒙特卡洛模拟缓存中筛选出满足条件的样本，然后计算筛选后Y变量的平均值，并通过弹窗展示结果。

## 核心逻辑

### 1. 缓存检查
```
用户点击"确定" 
  ↓
检查Y单元格是否有蒙特卡洛缓存
  ↓
如果没有缓存 → 弹窗提示"请先运行蒙特卡洛模拟" → 终止
  ↓
如果有缓存 → 继续执行压力测试
```

### 2. 压力测试执行
```
对每个X变量：
  施加下限压力 → 记录Y的响应
  施加上限压力 → 记录Y的响应
  ↓
将结果写入"压力测试结果"工作表
```

### 3. 条件筛选统计
```
从蒙特卡洛缓存中获取Y和所有X的样本数据
  ↓
对每个X变量：
  根据压力范围 [下限, 上限] 创建筛选条件
  保留X在压力范围内的样本
  ↓
合并所有X的筛选条件（AND逻辑）
  ↓
应用筛选掩码到Y数据
  ↓
计算筛选后Y的平均值
  ↓
弹窗显示结果
```

## 使用示例

### 示例1：基本使用

**场景**：
- Y单元格：D10（利润）
- X变量：
  - A1（销售量）：压力范围 ±20%
  - B2（成本）：压力范围 ±10%

**操作步骤**：
1. 先运行蒙特卡洛模拟（5000次迭代）
2. 打开压力测试配置窗口
3. 选择Y=D10，X=A1,B2
4. 设置压力范围
5. 点击"确定"

**结果**：
```
压力测试结果工作表：
- A1 下限: Y变化 -18.5%
- A1 上限: Y变化 +19.2%
- B2 下限: Y变化 +8.3%
- B2 上限: Y变化 -8.1%

弹窗显示：
条件筛选统计分析结果

因变量: D10
总样本数: 5000
筛选后样本数: 1250
筛选比例: 25.00%

筛选后因变量平均值: 105.3456
```

**解读**：
- 在5000次模拟中，有1250次（25%）同时满足：
  - A1在基准值的80%-120%范围内
  - B2在基准值的90%-110%范围内
- 在这1250次中，D10的平均值为105.35

### 示例2：筛选条件的含义

假设基准值：
- A1 = 100
- B2 = 50

压力范围设置：
- A1: ±20% → 筛选范围 [80, 120]
- B2: ±10% → 筛选范围 [45, 55]

筛选逻辑：
```python
# 从5000次模拟中筛选
mask_A1 = (A1_samples >= 80) & (A1_samples <= 120)
mask_B2 = (B2_samples >= 45) & (B2_samples <= 55)
combined_mask = mask_A1 & mask_B2

# 应用到Y
filtered_Y = Y_samples[combined_mask]
mean_Y = np.mean(filtered_Y)
```

#### 2. 条件筛选统计（L189-268）
```python
def _perform_conditional_analysis(cfg, y_cache):
    # 获取Y数据
    y_data = y_cache['data']
    
    # 初始化筛选掩码
    combined_mask = np.ones(total_samples, dtype=bool)
    
    # 对每个X应用筛选条件
    for xc in x_configs:
        x_data = get_cell_cache_info(x_addr)['data']
        
        # 计算压力范围
        x_min = calc_stressed_value(x_base, lo, hi, False)
        x_max = calc_stressed_value(x_base, lo, hi, True)
        
        # 创建筛选掩码
        x_mask = (x_data >= x_min) & (x_data <= x_max)
        combined_mask &= x_mask
    
    # 应用筛选
    filtered_y_data = y_data[combined_mask]
    y_mean = np.mean(filtered_y_data)
    
    # 弹窗显示
    xlcAlert(f"筛选后因变量平均值: {y_mean:.4f}")
```

## 优势

### 1. 自动化
- 无需手动配置筛选条件
- 压力范围自动转换为筛选条件
- 一键完成压力测试和条件统计

### 2. 直观性
- 压力范围即筛选范围
- 结果直接弹窗显示
- 清晰展示筛选比例

### 3. 实用性
- 回答"在压力范围内，Y的平均表现如何"
- 评估压力场景的代表性
- 辅助风险评估决策

## 应用场景

### 场景1：风险评估
**问题**：在销售量和成本都处于压力范围内时，利润的平均水平是多少？

**操作**：
- Y = 利润
- X1 = 销售量（±20%）
- X2 = 成本（±10%）

**结果**：筛选后利润平均值 = 95.2（低于基准值100）

**结论**：在压力条件下，利润平均下降4.8%

### 场景2：敏感性分析
**问题**：哪个变量的压力范围对结果影响更大？

**操作**：
- 测试1：只对X1施加压力
- 测试2：只对X2施加压力
- 测试3：同时对X1和X2施加压力

**对比**：
- 测试1筛选比例：50%，Y均值：98
- 测试2筛选比例：60%，Y均值：102
- 测试3筛选比例：30%，Y均值：95

**结论**：X1的影响更大

### 场景3：情景规划
**问题**：在最坏情况下（所有X都在下限），Y的平均值是多少？

**操作**：
- 设置所有X的压力范围为 [-20%, 0%]
- 只保留下限场景的样本

**结果**：筛选后Y均值 = 75

**结论**：最坏情况下，Y平均下降25%

## 注意事项

### 1. 缓存依赖
- **必须先运行蒙特卡洛模拟**
- 如果没有缓存，会弹窗提示并终止
- 清除模拟缓存后需要重新运行

### 2. 样本数量
- 筛选后样本数可能很少
- 如果筛选比例<5%，统计结果可能不可靠
- 建议增加模拟迭代次数或放宽压力范围

### 3. 数据匹配
- Y和所有X必须来自同一次模拟
- 样本数量必须一致
- 如果不匹配，该X会被跳过

### 4. 筛选逻辑
- 多个X条件之间是AND关系（同时满足）
- 压力范围是闭区间 [min, max]
- 百分比压力会转换为绝对值范围

## 扩展功能

### 可能的增强方向

1. **更多统计量**
   - 除了平均值，还可以显示标准差、分位数
   - 对比筛选前后的分布变化

3. **多种筛选模式**
   - 当前：AND逻辑（所有条件同时满足）

   - 可选：自定义组合逻辑

4. **导出筛选结果**
   - 将筛选后的样本导出到新工作表
   - 支持进一步分析

## 总结

条件筛选统计分析功能将压力测试和蒙特卡洛模拟有机结合：
- **压力测试**：评估极端场景的影响
- **条件筛选**：评估压力范围内的平均表现
- **统计分析**：量化风险和机会

这为风险管理和决策支持提供了强大的工具。
